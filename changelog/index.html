Changelog

Bug fixes, new features, and updates to Form and Field.

v11.0.0 March 2026 Latest
New Features
  • Gridlines panel — place numeric or alphabetic gridline systems on your model
  • CAD visibility independence — CAD overlays now have their own visibility state
  • Assembly beam inventory — auto-detect cross sections with board feet and lengths
  • Cadworks CSV import for assemblies
  • Smart Diff for multiverse model comparison
  • Visual Classifier — sort uncategorized entities by name, material, volume, or color
Improvements
  • SF Face edit persistence — edits to face measurements now save correctly
  • Base64 execute_script for improved SketchUp 2026 compatibility
  • Floating selection bar for quick bulk actions
  • Secondary filter bar with Material, Size, Tag, Type, and Function filters
Bug Fixes
  • Fixed isolation visibility not respecting locked categories
  • Fixed measurement cards not updating after combine operations
  • Fixed undo stack overflow on rapid bulk edits
v10.0.0 February 2026
New Features
  • Multiverse — model comparison with A/B views and quantity deltas
  • Cost Code Editor — full cost code library with CSV import/export
  • Category Templates — save and reuse classification templates across projects
  • Hyper Parse — sort entities by name, color, or grab-selected properties
  • Measurements panel with LF, SF, Volume, Count, and Wall tools
  • Parts Builder with 7 formula types (Sheet Goods, Linear Stock, Roll Goods, Wall Framing, On Center, Area Coverage, Custom Calc)
  • Assembly system with parts lists and CSV export
  • Project Notes with RFI, Punch, Field, Change, Safety types
Improvements
  • Tab-based dashboard with Takeoff, Measurements, Notes, Assemblies, Multiverse, CAD
  • EKG heartbeat indicator for backend connectivity status
  • Container-based grouping for Revit model organization
  • Undo support with up to 50 steps
